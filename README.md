# Blootie Django

Proyecto educativo infantil hecho con Python + Django usando el enfoque:
- Models
- Views
- Templates

## Instalación rápida
1. Abre terminal en esta carpeta.
2. Crea entorno virtual:
   python -m venv venv
3. Actívalo:
   Windows: venv\Scripts\activate
4. Instala dependencias:
   pip install -r requirements.txt
5. Ejecuta migraciones:
   python manage.py migrate
6. Carga datos demo:
   python manage.py seed_blootie
7. Ejecuta servidor:
   python manage.py runserver
